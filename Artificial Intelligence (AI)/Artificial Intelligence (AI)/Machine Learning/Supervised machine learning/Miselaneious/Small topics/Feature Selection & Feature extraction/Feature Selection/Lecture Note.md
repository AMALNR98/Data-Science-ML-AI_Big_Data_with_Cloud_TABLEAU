***Feature selection***
---
***Chi Square test***
--
- Input feature[categorical] va output feature[categorical]-
- non -ve values
---
- H0 : input features are not correlated to output feature
- H1 : correlated
- p-value > 0.05   -> accept
- p-value > 0.05   -> reject
- categorical  -> Nominal[r,g,b], ordinal[1st, 2nd, 3rd], binary[0,1]
